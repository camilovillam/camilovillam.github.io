# Opportunities and Tasks Tracker (Power Apps - Canvas App and Dataverse for Teams)

*Back to [Projects](https://camilovillam.github.io/#opportunities-and-tasks-tracker-power-apps---canvas-app-and-dataverse-for-teams).*

## Description

A responsive Canvas App using Dataverse for Teams as datasource for opportunities and tasks tracking and planning for a SMB. The goal is to manage, assign and visualize team tasks for different opportunities.

## Features

- Complete CRUD operation into Dataverse
- Personalized Home Screen for easy navigation
- Collapsible side menu
- Consistent look and feel, responsive layout
- Data grids and forms with tabbed menus for dynamic filtering, comments per record, and display of related records for easier navigation
- Visual display of tasks and opportunities using a collapsible Gantt diagram with dynamic zoom and filters
- Power Automate flows for Teams notifications


## Technologies and tools

- Power Apps (Canvas App)
- Dataverse for Teams
- Power Automate for notifications in Teams
